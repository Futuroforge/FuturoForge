# Launchpad IDE

A small Replit-style browser app builder.

## Features

- Browser code editor
- File list and tabs
- Run button with live preview
- Console output
- Keyword app builder: landing, dashboard, tracker, game
- Local storage save
- JSON project export
- Wide and phone preview modes

## Open

Open `index.html` in a browser, or serve this folder with a static server.
